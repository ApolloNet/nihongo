# Particules

- Sujet : は
- Aussi-sujet : も
- Objet : が
- Lieu : で
- Lieu : に
- Moyen : で
- Direction : へ
- Possesion, relation : の
- Et / avec (énumération pas forcément exhaustive mais quand-meme) : と
- Et / avec (énumération non-exhaustive) : や
- Complément d'objet : を
- Depuis : から
- Vers : まで
- Comparaison : より
- Interrogation : か
- Exclamation : よ
