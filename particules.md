# Particules

- Sujet : は
- Objet : が
- Lieu : で
- Lieu : に
- Moyen : で
- Direction : へ
- Possesion, relation : の
- Et / avec (nombre connu) : と
- Et / avec (nombre inconnu) : や
- Complément d'objet : を
- Depuis : から
- Vers : まで
- Comparaison : より
- Interrogation : か
- Exclamation : よ
